## Export 방법
1. 마우스로 좌표 클릭
2. Save Button 클릭
3. input 잘못 클릭시 delete input 클릭
4. 1~3 반복 후 완료 후 export 버튼 클릭 ( 콘솔창에서 복사, 콘솔창 여는 법:ctrl+shift+i )
